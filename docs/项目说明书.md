# 项目说明书

## 项目名称

锂离子电池热失控气体生成及可燃性评价数智化实验平台

## 项目目标

本项目用于化学实验教学中的虚拟仿真分析。系统通过公开文献归一化产气数据、可燃极限常数和虚拟空间场景，展示气体组成分析、混合可燃下限计算、空间浓度估算和教学报告生成流程。

## 使用范围

适用于：

- 化学实验课程中的数据分析教学。
- 新能源材料安全化学相关课程项目。
- 学生团队对虚拟仿真实验软件的后续维护和扩展。
- 软件登记材料整理中的功能说明和截图准备。

不适用于：

- 真实事故预测。
- 消防应急处置。
- 工程防爆设计。
- 真实热失控实验操作指导。

## 当前功能

- 首页驾驶舱
- 实验导学
- 文献数据库
- 虚拟实验
- 可燃极限计算
- 实验报告生成
- CSV 数据加载
- Plotly 图表展示
- 单元测试

## 技术路线

项目采用 Python 和 Streamlit 实现界面，使用 Pandas 读取和处理 CSV 数据，使用 Plotly 绘制交互图表。计算模块采用普通 Python 函数实现，当前不使用数据库、后端框架或机器学习模型。

## 安全说明

本平台仅用于化学实验教学、虚拟仿真和数据分析演示。平台不提供真实危险实验流程，计算结果不作为真实安全判据。
