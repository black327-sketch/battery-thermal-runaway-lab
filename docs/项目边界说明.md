# 项目边界说明

## 项目属于什么

本项目是面向化学实验教学的虚拟仿真平台。当前版本围绕公开文献归一化产气数据、混合可燃下限计算和教学报告生成展开。

项目可用于：

- 文献产气组成数据浏览。
- 可燃组分识别。
- Le Chatelier 混合规则教学演示。
- 虚拟空间浓度估算。
- 风险比值 R 的课堂讨论。
- 教学报告生成。

## 项目不属于什么

本项目不是：

- 电池爆炸预测软件。
- 消防应急系统。
- 工程防爆设计平台。
- 真实热失控实验系统。
- 可燃气体爆炸模拟器。

## 计算结果的使用边界

平台计算基于公开文献归一化数据、简化虚拟场景和教学模型。实际热失控过程中的温度、压力、气体演化速率、空间流场、惰性稀释和局部积聚均未被完整描述。

因此，页面和报告中的结果仅用于教学演示和模型理解，不作为真实安全评价或工程决策依据。

## 数据边界

当前数据仍需要补充完整来源信息。后续应补充文献题名、作者、年份、期刊、DOI、数据来源位置、LFL/UFL 常数来源和 CAS 号等材料。

## 安全边界

本平台不提供真实危险实验流程。学生和使用者不得根据页面、数据或报告内容尝试开展真实热失控实验、制备可燃气体、混合可燃气体或点火实验。
