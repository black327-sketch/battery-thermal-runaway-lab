# 软件功能说明

## 软件名称

锂离子电池热失控气体生成及可燃性评价数智化实验平台

## 软件版本

v0.2.1

## 软件定位

本软件是面向化学实验教学、虚拟仿真和数据分析演示的轻量化教学平台。平台围绕锂离子电池热失控公开文献产气组成数据，提供数据浏览、可燃组分识别、混合可燃下限计算、虚拟空间浓度估算、教学风险等级展示和实验报告生成等功能。

本平台仅用于化学实验教学、虚拟仿真和数据分析演示，不用于真实事故预测、消防应急或工程防爆设计。

## 主要功能模块

1. 首页驾驶舱
   - 展示软件名称、版本、定位、数据资产概览、功能模块入口、教学流程和软件特色。

2. 实验导学模块
   - 提供学习目标、教学路径、核心概念、计算模型说明、风险等级说明、模型假设与局限性、课前思考问题。

3. 文献数据库模块
   - 加载 CSV 文献样本数据，支持材料体系、SOC、触发方式和关键字筛选。
   - 展示样本详情、气体组成表、柱状图、环形图和组成总和检查。
   - 支持当前筛选结果 CSV 下载。

4. 虚拟实验模块
   - 支持选择文献样本、虚拟场景和虚拟总产气量。
   - 计算 LFL_mix、虚拟空间浓度、风险比值 R 和教学风险等级。
   - 支持场景对比和最近一次虚拟实验记录保存。

5. 可燃极限计算模块
   - 分步展示 Le Chatelier 混合规则计算过程。
   - 展示可燃组分识别、可燃气体总比例、归一化 y_i、LFL_i 和 y_i / LFL_i 贡献。

6. 实验报告生成模块
   - 自动读取最近一次虚拟实验记录或手动选择参数。
   - 生成结构化 Markdown / HTML 教学报告。
   - 报告包含实验目的、平台说明、安全边界、数据来源、计算过程、结果解释、模型局限性和思考题。

## 使用对象

- 化学实验课程学生
- 新能源材料安全化学方向教学人员
- 虚拟仿真实验课程建设人员
- 科研竞赛展示和初版软件作品整理人员

## 运行环境

- Python 3.10 或更高版本
- Streamlit
- Pandas
- Plotly
- CSV 文件数据

## 功能流程

数据选择 → 气体组成分析 → 可燃组分识别 → LFL_mix 计算 → 虚拟空间浓度估算 → R 风险教学评价 → 报告生成
