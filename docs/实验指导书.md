# 实验指导书

## 锂离子电池热失控过程中气体生成及可燃性评价 —— 虚拟实验

---

## 一、实验目的

1. 了解锂离子电池热失控过程中的主要产气种类及其来源
2. 掌握混合气体可燃极限（LFL）的基本概念和 Le Chatelier 混合规则
3. 学会使用虚拟实验平台进行混合气体可燃性评价
4. 理解风险比值 R 的物理含义及其在安全性评价中的作用
5. 体验数智化方法在化学实验教学中的应用

## 二、实验原理

### 2.1 锂离子电池热失控产气机理

锂离子电池在过充、过热或机械滥用条件下，电解液和电极材料发生热分解反应，
产生多种气体，主要包括：
- **H₂**：来源于电解液中微量水分的还原及 SEI 膜分解
- **CO、CO₂**：来源于碳酸酯类溶剂的热分解
- **CH₄、C₂H₄、C₂H₆**：来源于电解液有机组分的裂解反应
- **HF**：来源于 LiPF₆ 的热分解（不可燃但剧毒）

### 2.2 可燃极限（Flammability Limit）

可燃气体与空气混合时，只有在一定的浓度范围内才能被点燃：
- **可燃下限 (LFL)**：能够维持火焰传播的最低可燃气体浓度
- **可燃上限 (UFL)**：能够维持火焰传播的最高可燃气体浓度

### 2.3 Le Chatelier 混合规则

对于多组分混合可燃气体，其可燃下限可由 Le Chatelier 混合规则估算：

$$LFL_{mix} = \frac{100}{\sum_{i=1}^{n} \frac{c_i}{LFL_i}}$$

其中：
- $LFL_{mix}$：混合气体的可燃下限（% vol）
- $c_i$：组分 i 在可燃气体中所占的体积百分比（%）
- $LFL_i$：组分 i 的纯物质可燃下限（% vol）

### 2.4 风险比值 R

$$R = \frac{C_{当前}}{LFL_{mix}}$$

- 当 R < 1 时，当前浓度低于可燃下限，理论上不可点燃
- 当 R ≥ 1 时，当前浓度达到或超过可燃下限，存在点燃风险

## 三、实验流程

1. **文献数据查阅**：浏览平台内置的公开文献产气数据
2. **选择实验场景**：从预设的虚拟场景中选择实验条件
3. **设置实验参数**：调整虚拟空间体积、气体浓度等参数
4. **计算可燃极限**：运行 LFL_mix 计算
5. **风险评价**：计算 R 值并判定风险等级
6. **报告生成**：自动生成并导出实验报告

## 四、数据分析任务

1. 对比不同正极材料体系（NCM111、NCM523、LFP）的产气组成差异
2. 分析 SOC 对产气组成的影响
3. 计算不同场景下的 LFL_mix 并比较风险等级
4. 讨论 CO₂ 等不可燃气体对混合气体可燃性的惰性稀释效应

## 五、思考题

1. 为什么 H₂ 在热失控产气中通常占比最高？它对混合气体可燃性的贡献如何？
2. 如果虚拟空间的通风条件从"密闭"变为"正常通风"，风险比值 R 会如何变化？为什么？
3. Le Chatelier 混合规则的适用条件是什么？在什么情况下可能产生较大误差？
4. 为什么本平台采用虚拟仿真而非真实实验？这在化学实验教学中有何优势？
