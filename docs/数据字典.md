# 数据字典

本文件说明 `data/` 目录下 CSV 文件的用途、字段含义和维护要求。修改数据文件后，应运行 `pytest tests/ -v`，并在 Streamlit 页面中检查数据加载、筛选和报告生成功能。

## 1. normalized_gas_data.csv

用途：保存锂离子电池热失控产气组成的归一化样本数据，用于文献数据库、虚拟实验和可燃极限计算页面。

| 字段名 | 含义 | 单位 | 是否必填 | 示例值 | 维护说明 |
|---|---|---|---|---|---|
| `source` | 样本来源标识 | 无 | 是 | 示例文献A_NCM111_100SOC | 当前为简化标识；后续应补充真实文献题名、作者、年份、期刊和 DOI。 |
| `SOC_pct` | 荷电状态 | % | 建议必填 | 100 | 用于筛选和样本标签。 |
| `H2_pct` | 氢气体积分数 | % vol | 建议必填 | 30.2 | 可燃组分，参与 LFL_mix 计算。 |
| `CO_pct` | 一氧化碳体积分数 | % vol | 建议必填 | 25.1 | 可燃组分，参与 LFL_mix 计算。 |
| `CO2_pct` | 二氧化碳体积分数 | % vol | 建议必填 | 20.3 | 非可燃组分，不参与 Le Chatelier 求和。 |
| `CH4_pct` | 甲烷体积分数 | % vol | 建议必填 | 8.5 | 可燃组分，参与 LFL_mix 计算。 |
| `C2H4_pct` | 乙烯体积分数 | % vol | 建议必填 | 5.2 | 可燃组分，参与 LFL_mix 计算。 |
| `C2H6_pct` | 乙烷体积分数 | % vol | 建议必填 | 3.1 | 可燃组分，参与 LFL_mix 计算。 |
| `others_pct` | 其他气体体积分数 | % vol | 可选 | 7.6 | 不参与当前 LFL_mix 求和；如需细分，应新增明确字段并更新计算说明。 |
| `notes` | 样本备注 | 无 | 建议必填 | NCM111 体系 100% SOC 过充热失控 | 用于筛选触发方式和样本说明；不得写入真实危险操作流程。 |

维护注意：

- 气体组成字段应使用 `% vol`，字段名以 `_pct` 结尾。
- 各组分总和建议接近 100%，页面会显示组成总和检查。
- 当前数据缺少真实文献完整来源和可靠性等级，后续应补充到新增字段或单独数据表中。

## 2. gas_lfl_constants.csv

用途：保存气体可燃极限常数和可燃性标识，用于识别可燃组分并计算 LFL_mix。

| 字段名 | 含义 | 单位 | 是否必填 | 示例值 | 维护说明 |
|---|---|---|---|---|---|
| `gas_name` | 气体中文名称 | 无 | 是 | 氢气 | 使用中文通用名。 |
| `gas_formula` | 气体分子式 | 无 | 是 | H2 | 应与气体组成字段前缀一致。 |
| `LFL_vol_pct` | 可燃下限 | % vol | 可燃气体必填 | 4.0 | 单位为 % vol，不转换为 0.04。非可燃气体可留空。 |
| `UFL_vol_pct` | 可燃上限 | % vol | 可选 | 75.0 | 当前计算未使用，但建议保留。 |
| `is_combustible` | 是否可燃 | 无 | 是 | true | 只接受可明确识别的 true/false。 |
| `notes` | 备注 | 无 | 可选 | 教学计算常数 | 建议补充数据来源说明，当前不得编造来源。 |

维护注意：

- `is_combustible=false` 的气体不参与 LFL_mix 求和。
- LFL/UFL 数据来源需后续补充，例如数据手册、标准资料或权威数据库来源。
- 当前文件缺少 CAS 号和数据来源字段，见 `docs/待补充材料清单.md`。

## 2.1 lfl_reference_template.csv

用途：作为后续补录正式 LFL/UFL 来源信息的扩展模板。当前模板不参与程序计算，不替代 `gas_lfl_constants.csv`。

| 字段名 | 含义 | 单位 | 是否必填 | 示例值 | 维护说明 |
|---|---|---|---|---|---|
| `gas_id` | 气体编号 | 无 | 建议必填 | 待补充 | 可使用项目内部编号。 |
| `gas_name_cn` | 中文名称 | 无 | 建议必填 | 待补充 | 不应编造。 |
| `gas_name_en` | 英文名称 | 无 | 可选 | 待补充 | 与来源资料保持一致。 |
| `formula` | 分子式 | 无 | 建议必填 | H2 | 后续可与 `gas_formula` 对齐。 |
| `CAS_number` | CAS 号 | 无 | 待补充 | 待补充 | 需提供来源。 |
| `is_flammable` | 是否可燃 | 无 | 建议必填 | true | 后续可与 `is_combustible` 对齐。 |
| `LFL_vol_percent` | 可燃下限 | % vol | 可燃气体必填 | 待补充 | 需注明条件和来源。 |
| `UFL_vol_percent` | 可燃上限 | % vol | 可选 | 待补充 | 需注明条件和来源。 |
| `molecular_weight` | 分子量 | g/mol | 可选 | 待补充 | 如使用需提供来源。 |
| `role_in_thermal_runaway` | 热失控产气角色 | 无 | 可选 | 待补充 | 用于教学说明。 |
| `source` | 数据来源 | 无 | 建议必填 | 待补充 | 不得编造。 |
| `source_type` | 来源类型 | 无 | 建议必填 | 待补充 | 如标准、手册、数据库、文献。 |
| `reliability_level` | 可靠性等级 | 无 | 建议必填 | 待核验 | 需定义等级口径。 |
| `note` | 备注 | 无 | 可选 | 待补充 | 说明适用条件或待核验事项。 |

维护注意：

- 模板当前只提供字段结构，不表示已有正式来源数据。
- 后续如将模板用于页面展示或计算，应先补充测试和字段映射逻辑。

## 3. virtual_scenarios.csv

用途：保存虚拟空间场景参数，用于估算空间气体浓度和比较不同场景下的 R 值。

| 字段名 | 含义 | 单位 | 是否必填 | 示例值 | 维护说明 |
|---|---|---|---|---|---|
| `scenario_id` | 场景标识 | 无 | 是 | S001_实验室通风橱 | 应保持唯一，便于页面显示和报告记录。 |
| `room_volume_m3` | 虚拟空间体积 | m³ | 是 | 2.0 | 必须大于 0。 |
| `temp_c` | 场景温度字段 | °C | 可选 | 25 | 当前仅展示，不参与计算。 |
| `pressure_kpa` | 场景压力字段 | kPa | 可选 | 101.3 | 当前仅展示，不参与计算。 |
| `ventilation` | 通风条件标签 | 无 | 是 | poor | 当前支持 `none`、`poor`、`normal`。 |
| `gas_total_vol_pct` | 场景参考浓度字段 | % vol | 可选 | 0.5 | 当前页面主要使用用户输入的虚拟总产气量。 |
| `description` | 场景说明 | 无 | 建议必填 | 模拟实验室通风橱内微量气体累积 | 描述应保持教学化，不写真实工程处置建议。 |

维护注意：

- `ventilation` 会映射为通风稀释因子：`none=1.0`、`poor=2.0`、`normal=5.0`。
- 当前场景为教学参数，不代表真实场所安全评价模型。
