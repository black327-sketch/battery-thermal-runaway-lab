# 后续实验模块需求草案

本文件记录后续计划方向，不表示当前版本已经实现。所有扩展仍应保持虚拟仿真教学定位，不加入真实危险实验流程。

## 1. 学生实验记录模块

计划记录：

- 学生选择的文献样本。
- 学生选择的虚拟场景。
- 输入的虚拟总产气量。
- LFL_mix、空间浓度、R 值和教学风险等级。
- 学生对结果的文字解释。

## 2. 实验任务单模块

计划支持：

- 教师预设任务目标。
- 学生按步骤完成数据选择、虚拟计算和报告生成。
- 生成任务完成记录。

## 3. 数据上传模块

计划支持：

- 用户上传自定义 CSV。
- 平台进行字段检查和数据质量提示。
- 上传数据仅用于教学演示。
- 当前不设计正式数据库写入；如后续需要，应先设计审核流程。

## 4. 数据来源管理模块

计划记录：

- 文献来源。
- DOI 或可追溯链接。
- 数据来源位置。
- 数据可靠性等级。
- 待核验状态。

## 5. 报告导出增强

当前支持：

- Markdown。
- HTML。

后续可评估：

- Word。
- PDF。

## 6. 教师评价模块

计划支持：

- 查看学生提交记录。
- 给出文字反馈。
- 标注报告中数据来源、计算过程和模型边界说明是否完整。

该模块不做自动评分，也不输出真实风险判断。
