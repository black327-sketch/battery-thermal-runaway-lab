# 系统模块说明

## 首页驾驶舱模块

文件：`app/main.py`

首页作为软件入口，展示平台名称、版本、软件定位、安全声明、数据资产概览、核心功能模块、教学流程图和软件特色。该模块用于用户快速理解系统结构，也可作为软件材料截图页面。

## 实验导学模块

文件：`app/pages/1_实验导学.py`

实验导学模块承担课程导学和实验前培训功能。页面包含学习目标、教学路径、核心概念、Le Chatelier 混合规则、虚拟空间浓度估算、风险比值 R、风险等级说明、模型局限性和课前思考问题。

## 文献数据导入模块

文件：`app/pages/2_文献数据导入.py`

文献数据导入模块用于上传论文整理 CSV，按照模板检查字段、来源类型、文献定位和待补充状态。通过校验并确认后，可保存到 `data/experiment/validated/` 供二维实验台选择。

## 文献数据库模块

文件：`app/pages/3_文献数据库.py`

文献数据库模块用于展示项目 CSV 中的归一化产气组成数据。模块支持字段筛选、关键字搜索、数据表格展示、样本详情、气体组成柱状图、环形图、可燃/不可燃组分区分、组成总和检查和筛选结果下载。

## 虚拟实验模块

文件：`app/pages/4_虚拟实验.py`

虚拟实验模块提供参数控制区和结果看板。用户选择文献样本、虚拟场景和虚拟总产气量后，系统计算 LFL_mix、虚拟空间浓度、风险比值 R 和教学风险等级，并生成本次虚拟实验记录摘要。该记录会保存到 `st.session_state`，供报告生成模块默认读取。

## 二维交互实验台模块

文件：`app/pages/5_二维交互实验台.py`

二维交互实验台模块提供教学 ARC 模式和文献装置模式两条独立流程。页面包含步骤轴、二维装置图、仪器控制屏、分组按钮、DAQ 监控、实验日志、GC 图表、产气量记录、LFL_mix 可燃风险评价和报告状态。

## 可燃极限计算模块

文件：`app/pages/6_可燃极限计算.py`

该模块用于解释 Le Chatelier 混合规则的计算过程。页面从原始气体组成开始，依次展示可燃组分识别、可燃气体总比例、可燃组分归一化、LFL_i 常数、y_i / LFL_i 贡献项和 LFL_mix 结果。

## 实验报告生成模块

文件：`app/pages/7_实验报告生成.py`

报告模块用于生成正式教学报告。页面可读取最近一次虚拟实验记录，也可手动选择样本和场景。报告生成逻辑位于 `app/utils/report_generator.py`，支持 Markdown 和 HTML 两种输出格式。

## 工具函数模块

文件目录：`app/utils/`

- `app_config.py`：软件名称、版本、定位、安全声明、模块配置和风险颜色配置。
- `ui_theme.py`：全局 CSS 主题、视觉令牌和页脚。
- `ui_components.py`：页面标题、指标卡、说明卡、风险标签、步骤条等公共 UI 组件。
- `chart_utils.py`：Plotly 图表函数，包括气体组成、可燃占比、风险仪表盘、场景对比和 LFL 贡献图。
- `data_loader.py`：CSV 数据加载和文献样本筛选。
- `lfl_calculator.py`：LFL 常数读取、可燃组分比例、归一化和 LFL_mix 计算。
- `risk_model.py`：虚拟空间浓度、风险比值 R 和教学风险等级计算。
- `report_generator.py`：结构化 Markdown / HTML 教学报告生成。
