# 界面改造说明

## 本次改造目标

本次改造目标是将原 Streamlit 原型升级为具有初版软件作品形态的实验教学平台，使其更适合课堂演示、软件材料截图和后续功能扩展。

## UI 风格

界面采用科技感、实验室、能源安全和数据驾驶舱风格。主色使用深蓝、科技蓝和青蓝，辅色使用绿色、黄色、橙色和红色表示不同教学风险等级。页面背景采用浅灰蓝视觉基底，卡片使用浅边框、轻阴影和统一圆角。

## 页面结构优化

1. 首页改造为软件驾驶舱，突出软件名称、版本、定位、数据资产、功能模块和教学流程。
2. 实验导学页改造为课程导学和实验前培训页面，强调学习目标、模型说明和安全边界。
3. 文献数据库页改造为数据资产管理与可视化看板，增强筛选、搜索、详情和下载能力。
4. 虚拟实验页改造为左侧参数控制区和右侧结果看板，增强操作台观感。
5. 可燃极限计算页改造为可解释计算教学页，突出分步计算过程。
6. 实验报告页改造为正式教学报告生成模块，支持报告预览和下载。

## 图表优化

本次改造优先使用 Plotly 图表，新增或统一了以下图表：

- 气体组成柱状图
- 气体组成环形图
- 可燃 / 不可燃组分占比图
- 风险 R 值教学仪表盘
- 不同虚拟场景 R 值对比图
- y_i / LFL_i 贡献柱状图

图表统一中文标题、单位、hover 信息、颜色逻辑和空数据提示。

## 交互优化

1. 文献数据库支持材料体系、SOC、触发方式和关键字组合筛选。
2. 文献数据库支持当前筛选结果 CSV 下载。
3. 样本详情增加组成总和检查，便于发现异常或归一化差异。
4. 虚拟实验支持执行计算按钮，并将最近一次实验记录保存到 `st.session_state`。
5. 报告生成页可自动读取最近一次虚拟实验记录作为默认值。
6. 报告下载支持 Markdown 和 HTML 两种格式。

## 安全边界强化

所有主要页面和报告均保留或强化以下声明：

本平台仅用于化学实验教学、虚拟仿真和数据分析演示，不用于真实事故预测、消防应急或工程防爆设计。

## 2026-06-13 稳定性修复

1. 侧边栏页面顺序调整为：首页、实验导学、文献数据导入、文献数据库、虚拟实验、二维交互实验台、可燃极限计算、实验报告生成。
2. 二维交互实验台移除 HTML 假按钮和 `onclick` 交互，统一使用 Streamlit 原生按钮触发状态机。
3. 主题状态统一保存在 `st.session_state["theme_mode"]`，并新增 `.streamlit/config.toml` 作为稳定基础主题。
4. 新增安全图片渲染 helper，8 张机制图按首页、导学、二维实验台、文献数据库和可燃极限计算页面分区展示。
5. ARC 模式和文献装置模式重置时清理各自派生结果，避免旧的 LFL 风险评价、产气量、GC、DAQ 和报告状态残留。
