# 数据质量检查清单

修改或新增 CSV 数据后，请按本清单检查。

## 文献样本数据

- [ ] `source` 字段可以追溯到具体文献或数据来源。
- [ ] 已记录文献题名、作者、年份、期刊和 DOI（如有）。
- [ ] `SOC_pct` 单位为 `%`，取值可解释。
- [ ] 气体组成字段使用 `% vol`。
- [ ] 各气体组成之和接近 100%，或已说明归一化差异。
- [ ] `notes` 字段不包含真实危险实验操作步骤。
- [ ] 待核验数据已明确标注，不混同为已验证数据。

## LFL/UFL 常数

- [ ] LFL/UFL 单位统一为 `% vol`。
- [ ] 可燃气体的 `is_combustible` 为 `true`。
- [ ] 非可燃气体的 `is_combustible` 为 `false`，且不参与 LFL_mix 求和。
- [ ] 数据来源已记录，来源可追溯。
- [ ] 如补充 CAS 号，格式一致。
- [ ] 如使用 `lfl_reference_template.csv`，确认 `source`、`source_type`、`reliability_level` 均已填写或明确标记为待核验。

## 虚拟场景数据

- [ ] `scenario_id` 唯一。
- [ ] `room_volume_m3` 大于 0。
- [ ] `ventilation` 只使用当前支持的标签，或同步更新映射逻辑。
- [ ] 场景描述保持教学用途，不写工程防爆或应急处置建议。

## 修改后验证

- [ ] 运行 `pytest tests/ -v`。
- [ ] 运行或调用 `app/utils/data_quality.py` 中的数据质量检查函数。
- [ ] 启动 `streamlit run app/main.py`。
- [ ] 检查文献数据库页面能加载数据。
- [ ] 检查虚拟实验页面能完成计算。
- [ ] 检查可燃极限计算页面能显示贡献项。
- [ ] 检查实验报告能预览和下载。
