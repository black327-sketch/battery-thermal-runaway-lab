# 用户使用说明

## 启动平台

在 PowerShell 或命令行中运行：

```bash
cd D:\code\digital_and_intelligent_platform
streamlit run app/main.py
```

浏览器打开终端提示的地址，通常为：

```text
http://localhost:8501
```

## 推荐使用流程

1. 打开首页，了解平台定位和功能模块。
2. 进入“实验导学”，阅读学习目标、核心概念和安全边界。
3. 进入“文献数据库”，筛选或查看样本数据。
4. 进入“虚拟实验”，选择样本、场景和虚拟总产气量，执行计算。
5. 进入“可燃极限计算”，查看 LFL_mix 的分步计算过程。
6. 进入“实验报告生成”，预览并下载 Markdown 或 HTML 报告。

## 页面说明

- 首页：查看软件基本信息和数据资产概览。
- 实验导学：了解课程目标、模型假设和风险等级含义。
- 文献数据库：查看 CSV 样本、筛选字段、下载筛选结果。
- 虚拟实验：计算 LFL_mix、空间浓度和 R 值。
- 可燃极限计算：复核可燃组分归一化和贡献项。
- 实验报告生成：生成教学报告。

## 常见问题

### 页面打不开

确认是否已经运行：

```bash
streamlit run app/main.py
```

如果 8501 端口被占用，请使用终端提示的新地址。

### 数据表为空

检查 `data/` 目录下 CSV 文件是否存在，字段名是否与 `docs/数据字典.md` 一致。

### 无法计算 LFL_mix

可能原因包括：所选样本缺少可燃组分、LFL 常数表缺少对应气体、数据字段为空或非数值。

### 没有先运行虚拟实验，能否生成报告

可以。报告页面支持手动选择样本、虚拟场景和总产气量。

## 安全边界

本平台仅用于化学实验教学、虚拟仿真和数据分析演示，不用于真实事故预测、消防应急或工程防爆设计。用户不得根据本平台内容开展真实危险实验。
