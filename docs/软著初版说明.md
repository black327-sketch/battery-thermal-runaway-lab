# 软著初版说明

## 当前版本性质

当前版本为“锂离子电池热失控气体生成及可燃性评价数智化实验平台”的初版教学软件。该版本已经具备 Streamlit 多页面软件形态、统一界面风格、模块化代码结构、数据加载、计算展示、虚拟实验和报告生成闭环。

## 已实现功能

1. 软件驾驶舱首页
2. 实验导学与实验前培训
3. 文献数据筛选、搜索、展示和下载
4. 虚拟实验参数控制和结果看板
5. 混合可燃下限 LFL_mix 分步计算展示
6. 风险比值 R 教学评价
7. 最近一次虚拟实验记录保存
8. Markdown / HTML 教学报告生成
9. 公共 UI 主题、组件和图表工具
10. 软件功能、模块、界面改造和后续规划文档

## 后续计划功能

后续计划功能包括更丰富的文献字段管理、学生学习记录、教师评价、报告模板扩展、课程任务单和更多教学可视化图表。上述内容为计划或预留方向，不代表当前版本已经完成。

## 软件安全边界

本平台仅用于化学实验教学、虚拟仿真和数据分析演示，不用于真实事故预测、消防应急或工程防爆设计。

当前版本不包含真实热失控实验、过充、针刺、加热、点火、制备可燃气体、混合可燃气体或工程防爆设计相关流程。所有风险等级均为教学评价标签，不作为真实工程安全判据。

## 软件运行方式

```bash
cd D:\code\digital_and_intelligent_platform
pip install -r requirements.txt
streamlit run app/main.py
```

## 测试方式

```bash

pytest tests/ -v
```
