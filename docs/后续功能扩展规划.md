# 后续功能扩展规划

以下内容为后续计划或预留方向，不代表当前版本已经实现。

## 后续实验仿真模块扩展

计划在保持安全边界的前提下，扩展更多虚拟场景参数组合、课程任务单和教学变量对比功能。所有新增内容仍应保持虚拟仿真教学定位，不加入真实危险实验流程、装置搭建或操作步骤。

## 数据管理扩展

计划补充更规范的文献元数据字段，例如文献标题、年份、材料体系、SOC、测试条件摘要和数据来源说明。新增字段应来自公开资料或用户自行整理，不编造 DOI、作者、实验结果或真实验证信息。

## 学生学习记录扩展

计划增加本地化的学习记录导出功能，例如记录学生选择的样本、场景、计算结果、思考题回答和报告生成时间。当前版本未引入数据库，后续也可优先采用 CSV 或 JSON 文件导出方式。

## 教师评价功能扩展

计划增加教师评价维度，例如数据理解、计算过程、模型局限性说明、报告完整性和安全边界表述等评分项。该功能应定位为教学评价，不作为工程安全评价。

## 报告模板扩展

计划增加多套报告模板，例如课程作业模板、竞赛展示模板、实验指导书附录模板和学生学习档案模板。模板内容应继续保留虚拟仿真教学声明和安全边界声明。

## 图表与可解释性扩展

计划扩展更多可解释图表，例如不同样本 LFL_mix 对比、可燃组分贡献雷达图、文献样本分类统计图和课程任务进度图。图表应服务于教学理解，不追求与真实事故预测相关的复杂模拟。

## 注意事项

后续扩展不得加入真实热失控实验操作步骤、可燃气体制备或混合流程、点火流程、事故预测结论、消防应急建议或工程防爆设计建议。
