# AI_ASSISTANT_REPORT

更新时间：2026-06-19 12:08

## 本轮结论

AI 学伴已从“功能面板”继续收敛为桌宠式即时聊天组件。主界面只突出聊天，不再默认铺开模式说明、历史记录、评价、导出和调试信息。

## 当前体验

- 默认收起：只显示可拖拽电池桌宠，状态通过小圆点表达。
- 展开后：标题栏、在线状态点、最小化、关闭、`⋯` 更多菜单、回答区、输入区。
- 空闲欢迎语：`你好，我可以陪你完成实验、解释原理，也可以回答其他学习问题。`
- 快捷入口最多三个：下一步、报警解释、实验现象解释类问题。
- 重复提交同一问题会被拦截。
- 请求中显示轻量加载状态。

## 移入更多菜单的功能

- 查看历史记录
- 生成学习评价
- 下载评价 Markdown
- 导出学习记录 JSON
- 查看当前模式
- 结束本轮对话
- 清空历史
- 重新开始实验

## DeepSeek 与本地兜底

- 成功时仍显示 `DeepSeek 智能伴学模式`。
- DeepSeek 失败时主界面不铺大段技术信息，只短暂 toast 提示。
- 本地规则兜底保留。
- Key 只在 Streamlit 后端读取，不进入前端 JavaScript、截图、报告或下载文件。
- 适配器新增 SSE 流式解析能力，当前浮窗桥接保持受控后端请求以稳定实验状态。

## 上下文控制

传给 DeepSeek 的上下文保持精简：

- 当前问题
- 当前页面
- 当前 SOC
- 当前阶段
- trace 计数摘要
- 当前过程建议摘要
- 精简事实摘要和安全边界

不传完整聊天记录、完整操作日志、整篇参考文献或 API Key。

## 状态保护

- AI 提问不会重置实验进度。
- 新增 localStorage 进度快照。
- 刷新后如检测到未完成实验，会提示“检测到未完成实验，是否恢复进度？”。
- 恢复必须由用户确认。
